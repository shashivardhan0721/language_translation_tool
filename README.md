The Language Translation Tool is a Python application that translates text from one language to another. It provides a simple and user-friendly interface where users can enter text, select source and target languages, and receive instant translations.

## ✨ Features
- Translate text between multiple languages
- Simple and easy-to-use interface
- Fast and accurate translations
- Supports multiple language pairs
- Lightweight Python application

## 🛠️ Technologies Used
- Python
- Tkinter
- Googletrans

language_translation_tool/

## 🚀 Installation

1. Clone the repository:

bash
git clone https://github.com/shashivardhan0721/language_translation_tool.git


2. Navigate to the project folder:

bash
cd language_translation_tool


3. Install the required package:

bash
pip install googletrans==4.0.0rc1


4. Run the application:

bash
python app.py


## 💻 Usage

1. Enter the text you want to translate.
2. Choose the source language.
3. Choose the target language.
4. Click the *Translate* button.
5. View the translated result.
